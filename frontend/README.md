# Common Ground Finder

An AI-powered application to help groups find the optimal time and location for meetups.

## Features
- **Preferences Collection**: Users submit availability, interests, and location.
- **AI Optimization**: Calculates the best time and venue using a weighted fairness algorithm.
- **Gemini Integration**: Provides natural language explanations for the "Why" of the decision.
- **Dynamic Results**: Visualization of attendance, travel distance, and alternative options.

## Tech Stack
- Frontend: React, Vite, Tailwind CSS, Shadcn UI
- Backend: Node.js, Express, Firebase Admin
- AI: Google Gemini API

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm start
```
